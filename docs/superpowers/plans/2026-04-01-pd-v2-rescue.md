# PD V2 Rescue — Salvaging Generalizable Improvements

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Investigate three rescue lines from the PD V2 research to find improvements that generalize across the full PDF corpus, not just ART_674.

**Architecture:** Single new sweep script (`sweep_rescue.py`) runs all three investigations with a shared cross-validation harness over 27 PDFs (22 general + 5 ART family). Each rescue line produces per-PDF count-only metrics (error, abs_error) compared to V1-count baseline. A final report summarizes which lines (if any) beat V1 on cross-validation MAE.

**Tech Stack:** numpy, cv2 (Canny for edge_density), existing `eval/pixel_density/` modules (features.py, metrics.py, cache.py, evaluate.py)

---

## Context

**Why this plan exists:** PD V2 (multi-descriptor bilateral: dark_ratio + edge_density, robust-z, KMeans) achieved F1=0.957 on ART_674 but **failed cross-validation** — MAE=22.4 vs V1-count MAE=20.3 on 21 general PDFs. V2's KMeans threshold overfits to the score distribution of large PDFs. However, individual components showed promise and may generalize if combined with V1's stable percentile threshold.

**The three rescue lines:**

1. **Rescue A — Edge density standalone with V1 threshold:** `edge_density_grid` alone hit F1=0.928 on ART_674 with pct_75.2. If it also works on other PDFs, it's a drop-in improvement. Hypothesis: the value is in the feature, not the threshold.

2. **Rescue B — Score fusion with V1 as base:** Use V1 scores (dark_ratio L2) as the dominant signal and blend in a small amount of edge_density scores. The V1 threshold (pct_75.2) stays the same. Hypothesis: a small edge boost improves recall without destabilizing V1.

3. **Rescue C — Multi-descriptor with V1 threshold:** The full dark_ratio + edge_density combo from V2, but replacing KMeans with pct_75.2. The robust-z normalization stays. Hypothesis: V2 failed because of KMeans, not because of the features or normalization.

**Success criteria:** A rescue line is viable if it achieves **lower MAE than V1-count on the 22-PDF general corpus** AND **lower MAE than V1-count on the 5-PDF ART family** (V1 baselines computed by the sweep itself — not hardcoded), OR if it achieves **comparable MAE with demonstrably better page-level accuracy on ART_674** (F1 > 0.922).

**Existing code to reuse:**
- `eval/pixel_density/features.py` — `feat_dark_ratio_grid()`, `feat_edge_density_grid()`, `extract_features()`

```python
from eval.pixel_density.features import feat_dark_ratio_grid  # noqa: E402
from eval.pixel_density.metrics import bilateral_l2  # noqa: E402


def _percentile_threshold(scores: np.ndarray, pct: float) -> list[int]:
    """Threshold at given percentile, ensuring page 0 is always included."""
    thresh = np.percentile(scores, pct)
    matches = [i for i in range(len(scores)) if scores[i] >= thresh]
    if 0 not in matches:
        matches.insert(0, 0)
    return matches


def scorer_v1(pages: np.ndarray, pct: float = 75.2) -> list[int]:
    """V1-count baseline: dark_ratio_grid 8x8, L2 bilateral, min, pct_75.2.

    Note: This uses the raw page arrays directly (no CLAHE preprocessing),
    matching the behavior of compute_variant_vectors("clahe") which applies
    CLAHE to each page. For exact V1 reproduction we apply CLAHE here too.

    Args:
        pages: Array of shape (N, H, W), uint8 grayscale pages.
        pct: Percentile threshold (default: 75.2).

    Returns:
        List of detected cover page indices (0-based).
    """
    import cv2

    clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
    vectors = []
    for i in range(pages.shape[0]):
        enhanced = clahe.apply(pages[i])
        vectors.append(feat_dark_ratio_grid(enhanced, grid_n=8))
    scores = bilateral_l2(vectors, "min")
    return _percentile_threshold(scores, pct)
```

```python
def _normalize_01(scores: np.ndarray) -> np.ndarray:
    """Min-max normalize scores to [0, 1]."""
    mn, mx = scores.min(), scores.max()
    if mx - mn < 1e-12:
        return np.zeros_like(scores)
    return (scores - mn) / (mx - mn)


def scorer_rescue_b(
    pages: np.ndarray,
    edge_weight: float = 0.2,
    pct: float = 75.2,
) -> list[int]:
    """Rescue B: V1 base + edge_density boost, fused scores with pct threshold.

    Args:
        pages: Array of shape (N, H, W), uint8 grayscale pages.
        edge_weight: Weight for edge_density scores (V1 gets 1 - edge_weight).
        pct: Percentile threshold (default: 75.2).

    Returns:
        List of detected cover page indices (0-based).
    """
    import cv2

    v1_weight = 1.0 - edge_weight

    # V1 scores (CLAHE + dark_ratio)
    clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
    v1_vectors = []
    for i in range(pages.shape[0]):
        enhanced = clahe.apply(pages[i])
        v1_vectors.append(feat_dark_ratio_grid(enhanced, grid_n=8))
    v1_scores = _normalize_01(bilateral_l2(v1_vectors, "min"))

    # Edge density scores
    edge_vectors = [feat_edge_density_grid(pages[i], grid_n=4) for i in range(pages.shape[0])]
    edge_scores = _normalize_01(bilateral_l2(edge_vectors, "min"))

    # Fuse
    fused = v1_weight * v1_scores + edge_weight * edge_scores
    return _percentile_threshold(fused, pct)
```

```python
from eval.pixel_density.features import extract_features  # noqa: E402


def _robust_z_normalize(matrix: np.ndarray) -> np.ndarray:
    """Robust z-score normalization using median and MAD.

    Args:
        matrix: 2-D array (n_samples, n_features).

    Returns:
        Normalized matrix, same shape.
    """
    median = np.median(matrix, axis=0)
    mad = np.median(np.abs(matrix - median), axis=0)
    mad[mad < 1e-12] = 1.0
    return (matrix - median) / (mad * 1.4826)


def scorer_rescue_c(pages: np.ndarray, pct: float = 75.2) -> list[int]:
    """Rescue C: dark_ratio + edge_density, robust-z norm, L2 bilateral, pct_75.2.

    Same feature combination as V2, but with V1's percentile threshold
    instead of KMeans.

    Args:
        pages: Array of shape (N, H, W), uint8 grayscale pages.
        pct: Percentile threshold (default: 75.2).

    Returns:
        List of detected cover page indices (0-based).
    """
    feat_list = ["dark_ratio_grid", "edge_density_grid"]
    vectors = [extract_features(pages[i], feat_list) for i in range(pages.shape[0])]
    matrix = np.vstack(vectors)
    normed = _robust_z_normalize(matrix)
    normed_list = [normed[i] for i in range(normed.shape[0])]
    scores = bilateral_l2(normed_list, "min")
    return _percentile_threshold(scores, pct)
```

```python
def compute_summary(results: list[dict]) -> dict:
    """Compute aggregate metrics from cross-validation results.

    Args:
        results: List of per-PDF result dicts with abs_error key.

    Returns:
        Dict with mae, exact (count), within_2 (count), n.
    """
    n = len(results)
    errors = [r["abs_error"] for r in results]
    return {
        "mae": sum(errors) / n if n else 0.0,
        "exact": sum(1 for e in errors if e == 0),
        "within_2": sum(1 for e in errors if e <= 2),
        "n": n,
    }


def format_comparison_table(
    v1_results: list[dict],
    rescue_results: dict[str, list[dict]],
) -> None:
    """Print a per-PDF comparison table: V1 vs each rescue line.

    Args:
        v1_results: V1-count per-PDF results.
        rescue_results: {rescue_name: per-PDF results}.
    """
    rescue_names = sorted(rescue_results.keys())
    # Header
    hdr = f"{'PDF':<14} {'Tgt':>4} | {'V1':>4} {'err':>5}"
    for rn in rescue_names:
        hdr += f" | {rn:>4} {'err':>5}"
    print(hdr)
    print("-" * len(hdr))

    # Rows
    rescue_by_name = {
        rn: {r["name"]: r for r in results}
        for rn, results in rescue_results.items()
    }

    for v1r in v1_results:
        name = v1r["name"]
        row = (f"{name:<14} {v1r['target']:>4} | "
               f"{v1r['matches']:>4} {v1r['error']:>+5}")
        for rn in rescue_names:
            rr = rescue_by_name[rn].get(name)
            if rr:
                row += f" | {rr['matches']:>4} {rr['error']:>+5}"
            else:
                row += f" | {'--':>4} {'--':>5}"
        print(row)

    # Summary
    print("-" * len(hdr))
    v1_s = compute_summary(v1_results)
    summ = f"{'MAE':<14} {'':>4} | {'':>4} {v1_s['mae']:>5.1f}"
    for rn in rescue_names:
        rs = compute_summary(rescue_results[rn])
        summ += f" | {'':>4} {rs['mae']:>5.1f}"
    print(summ)

    exact = f"{'Exact':<14} {'':>4} | {'':>4} {v1_s['exact']:>5}"
    for rn in rescue_names:
        rs = compute_summary(rescue_results[rn])
        exact += f" | {'':>4} {rs['exact']:>5}"
    print(exact)

    within = f"{'Within +/-2':<14} {'':>4} | {'':>4} {v1_s['within_2']:>5}"
    for rn in rescue_names:
        rs = compute_summary(rescue_results[rn])
        within += f" | {'':>4} {rs['within_2']:>5}"
    print(within)
```
