# frontend/

React + Vite UI for PDFoverseer. Components under `src/components/`, hooks
under `src/hooks/`, Zustand store under `src/store/`, shared primitives under
`src/ui/`.

Always use the `po-*` design tokens in JSX (`tailwind.config.js`) — never raw
`bg-slate-*`/`bg-indigo-*`/etc. See the root `README.md` and `CLAUDE.md` for
the project's architecture and conventions.
